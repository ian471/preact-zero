# 0️⃣ preact-zero

This is a template you can use to develop a web app using
[Preact](https://preactjs.com/) that uses [Snowpack](https://www.snowpack.dev/)
for unbundled development.

```
npm install
npx snowpack dev
```

## Build and Bundle (optimized)

```
npm install
npx snowback build
npx serve build
```
